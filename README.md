Repository Description

  This repository contains the supplementary material, datasets, simulation files, and supporting documents used to obtain and validate 
  the results presented in the manuscript:
  
  "Effect on the Driving Range of an Electric Vehicle When Photovoltaic Systems are Integrated Into Its Bodywork."
  
  The repository provides access to the photovoltaic simulations, experimental measurements, theoretical calculations, 
  and aerodynamic analyses used throughout the research.

Repository Contents

1. SIMULACION_PVSOL.zip

  This file contains the energy generation simulations performed using PV*SOL Premium for the photovoltaic system integrated into the 
  electric vehicle.
  
  Contents include:
  
  Energy generation reports for different vehicle orientations:
    -North
    -South
    -East
    -West
  Simulation scenarios considering:
    -5% system losses
    -40% system losses
  Original PV*SOL Premium project files used to perform the simulations.
  
2. TRAZADOR_DE_CURVAS.zip

  This file contains the experimental measurements obtained using the photovoltaic curve tracer.
  
  The ZIP file includes the following folders:
  
    -TRAZADOR_BAUL
    -TRAZADOR_CAPO
    -TRAZADOR_LATERALES
    -TRAZADOR_TECHO
  
  Each folder contains CSV files with the measured electrical parameters for different vehicle speeds.
  
  Additionally, the folder:
  
    -SOMBRAS
  
  contains CSV files with power and cell temperature measurements obtained under partial shading conditions.

3. ANEXOS_PRUEBAS_EXPERIMENTALES_Y_TEORICOS.pdf

  This document contains the supplementary information associated with the experimental and theoretical analyses developed in the study.
  
  The document includes:
  
    -Experimental setup description.
    -Photovoltaic module specifications.
    -Irradiance measurements.
    -I–V and P–V curve acquisition procedures.
    -Cell temperature measurements.
    -Theoretical power calculations.
    -Validation of experimental and theoretical results.
    -Supporting figures and tables.
    
4. AERODINAMICA.zip

  This file contains the aerodynamic and thermal analyses used in the manuscript.
  
  Included files:
  
  -Coeficiente_convectivo.xlsx
  -Coeficiente_arrastre.xlsx
  
  These spreadsheets contain:
  
  Simulated drag coefficient values for different vehicle speeds.
  Correlation between air velocity and convective heat transfer coefficient.
  Supporting calculations employed in the thermal and aerodynamic analyses.
